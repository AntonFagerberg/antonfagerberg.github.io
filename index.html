About myself & this site

Welcome to my part of the internet! My name is Anton Fagerberg. I am a software developer living in Lund, Sweden where I'm currently working on my master thesis.

For the last five years, I've been studying Computer Science and Engineering at Lund Univeristy (LTH). My specialisation is software in systems: "to develop large program systems, advanced programming techniques and software engineering solutions".

I've also been working part time during 2014 as a consultant at JayWay where I've been doing web programming, mainly with Scala and JavaScript (and some relational databases). Before that, I worked at Bring where I developed logistics systems in Scala and Play Framework.

My current programming language of choice is Elixir. I'm a believer in functional programming and Elixir has a simplicity to it with many concepts which I really like.

Beside Elixir, I've recently been working mainly with Scala which I've enjoyed and JavaScript which I've a little enjoyed less. Over the years, I've been coding some things in Java, C++, Ruby, PHP, Assembler among other things. I've made an Android app and an iPhone app just for fun. Check out my projects to see some of the things which I've created.

I like Free Software and try to have openness as my general approach in life. I've created this site as a place for me to write what's on my mind, to show off some projects and talk a bit about myself. Check out the links in the menu above and for additional information or check out my GitHub page to see what I'm currently working on.

Besides software development, I enjoy tasting new beers, cooking & eating good food, watching cartoons, reading books and I probably follow too many TV-series.
